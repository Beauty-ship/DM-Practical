This repository contains a collection of Python programs based on fundamental concepts of discrete mathematics and data structures. It includes implementations for SET operations, RELATION properties using matrix representation (reflexive, symmetric, anti-symmetric, transitive), permutation generation (with and without repetition), brute-force solutions for equations of the form x₁ + x₂ + ... + xₙ = C, polynomial evaluation using arrays (including Horner’s method), and graph algorithms such as checking for a complete graph and computing in-degree and out-degree using adjacency matrices. Each program is written in a clear and structured manner to help students understand core logic and prepare effectively for exams and practicals.
